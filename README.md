## Blog App
A modern blog platform built with React, featuring full CRUD operations, responsive UI, and clean code architecture. The application integrates Redux for efficient state management, JWT authentication for secure user access, and Tailwind CSS for fast and customizable styling.

## The App is Up and Running at:

**[abhishek-blog-app.netlify.app](https://abhishek-blog-app.netlify.app/)**

## Setup Steps:

### Step 1:

git clone `https://github.com/Ad-Abhishek/blog-app.git`

### Step 2:

cd blog-app

### Step 3:

npm i

### Step 4:

npm run dev
